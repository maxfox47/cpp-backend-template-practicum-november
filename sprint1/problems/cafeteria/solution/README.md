# Решение задачи "Кафетерий"

## Описание

Асинхронная система приготовления хот-догов с использованием boost::asio.

## Структура проекта

- `main.cpp` - основная программа с тестами
- `cafeteria.h` - класс кафетерия для заказа хот-догов
- `ingredients.h` - классы ингредиентов (хлеб, сосиска, склад)
- `hotdog.h` - класс хот-дога
- `gascooker.h` - газовая плита с горелками
- `clock.h` - типы для работы со временем
- `result.h` - вспомогательный класс для обработки результатов

## Сборка

### С использованием Conan (рекомендуется)

```bash
conan install . --build=missing
mkdir build && cd build
cmake .. -DCMAKE_TOOLCHAIN_FILE=../conan_toolchain.cmake
cmake --build .
```

### Без Conan (если Boost установлен системно)

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

## Запуск

```bash
./cafeteria
```

## Особенности реализации

- Асинхронное приготовление хот-догов с использованием boost::asio
- Газовая плита с 8 горелками и очередью ожидания
- RAII-управление ресурсами через GasCookerLock
- Многопоточность с синхронизацией через std::latch
- Валидация времени приготовления ингредиентов
- Обработка ошибок через класс Result
